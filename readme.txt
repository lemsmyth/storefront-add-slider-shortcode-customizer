== Description ==
Allows you to add a slider shortcode to your Storefront Homepage from the customizer. Works with most slider plugins that create a shortcode (Meta Slider, Soliloquy, Revolution Slider, Slide Anything, LayerSlider).

Only will work with the Storefront theme either as a parent theme or a child theme.

To get the [Storefront theme](https://wordpress.org/themes/storefront/).

== Installation ==

#### Installing the plugins
1. In your WordPress admin dashboard, go to *Plugins > Add New*, search for **Storefront Add Slider Shortcode in Customizer** and clich "*Install now*"
1. Alternatively, download the plugin and upload the contents .zip file at *Plugins > Add New*, Click on *Upload Plugin* and chose the downloaded .zip file *Install Now*
2. Activate the plugin.
3. Go to *Appearance > Customize*, scroll down to *Homepage Settings*,
4. Enter your shortcode created by your favorite slider plugin, enter it into the *Slider Shortcode* field, be sure you include the [] square brackets.
5. With the other checkboxes you can chose to hide the homepage title, also can chose to have the slider contained by the content or if you want the slider the full window width. Add if the slider should show on all pages or just on the home page.
